# Examples

This folder contains a small curated set of visuals copied from the local
working environment to give public reviewers a quick sense of project outputs
without requiring local data or checkpoint setup.

Included assets:

- `training_loss.png`
- `training_f1.png`
- `inference_slice_9145.png`
- `inference_slice_10476.png`

These files are representative examples, not a comprehensive result archive.
